# JavaScript Algorithms

## Türkçe

Bu repo, JavaScript öğrenirken çözdüğüm algoritma sorularını ve çözümlerini bir arada toplamak için oluşturuldu.

Her algoritma kendi klasöründe bulunur ve çözüm kodu `solution.js` dosyasında yer alır.

### Amaç

- JavaScript bilgilerimi geliştirmek
- Algoritma çözme pratiği yapmak
- Çözümlerimi düzenli bir şekilde paylaşmak

### Çalıştırma

Bilgisayarınızda Node.js kuruluysa istediğiniz çözümü terminalden çalıştırabilirsiniz:

```bash
node folder-name/solution.js
```

Örnek:

```bash
node factorial/solution.js
```

## English

This repository contains the algorithm problems I solve while learning JavaScript and their solutions.

Each algorithm is stored in its own folder, with the solution inside a `solution.js` file.

### Purpose

- Improve my JavaScript skills
- Practice problem-solving and algorithms
- Share my solutions in an organized way

### Running a Solution

If Node.js is installed, you can run any solution from the terminal:

```bash
node folder-name/solution.js
```

Example:

```bash
node factorial/solution.js
```
